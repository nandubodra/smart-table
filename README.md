
# 🍽️ SmartTable - Smart Restaurant Ordering System

A modern and interactive **QR-based restaurant ordering web app** where customers can browse menu items, add orders, and place requests directly from their table.

---

## 🚀 Features

* 📱 QR Code Based Ordering System
* 🧾 Live Order Summary Panel
* 🍔 Category-wise Menu (Appetizers, Main Course, Desserts, Beverages)
* ➕ Add to Cart Functionality
* 💰 Auto Price Calculation (Subtotal, Tax, Total)
* 🎨 Clean & Premium UI Design
* 🔍 Search Items Feature
* 🎁 Special Offers Section
* 📞 Call Waiter Button
* 💳 Pay Now Button (UI)

---

## 🖥️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
